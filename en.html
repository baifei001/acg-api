ACG API

Random Image API - JSON responses & client-side redirects

Quick Start

Get a random image as JSON:

curl https://username.github.io/acg-api/api/random

Jump directly to a random image:

https://username.github.io/acg-api/img/random

API Endpoints

GET /api/random
Return a random image as JSON. Add ?category=anime to filter by category, &size=5 for multiple.
GET /api/categories
Return all categories with image counts.
GET /img/random
Redirect to a random image. Add ?category=anime to filter.
